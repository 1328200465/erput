## Erupt

#### Author 
YuePeng<erupts@126.com>

#### 设计理念
对象视图映射 OVM(Object View Mapping)

#### 项目介绍
基于注解理念的java后台快速开发框架，前端页面结构完全由注解指定，将前端工作量减少至零
面向注解开发的通用数据管理框架

#### 软件架构
后端
1. Java8
2. Spring Boot
3. JPA

前端
1. Angular 8
2. ng-zorro
3. ng-alain
