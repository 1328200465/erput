### 设计理念

### 项目介绍
基于注解理念的java后台快速开发框架，前端页面结构完全由注解指定，将前端工作量减少至零
面向注解开发的通用数据管理框架

### 软件架构
后端
- Java8
- Spring Boot
- JPA

前端
- Angular 8
- ng-zorro
- ng-alain
